### 题目
https://leetcode.com/problems/missing-number/

### 分类
* 哈希表

### 答案
```
public int missingNumber(int[] nums) {

    Map<Integer, Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        map.put(nums[i], nums[i]);
    }
    for (int i = 0; i <= nums.length; i++) {
        if (!map.containsKey(i)) {
            return i;
        }
    }
    return -1;
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(n)
