### 题目
https://leetcode.com/problems/n-th-tribonacci-number/

### 分类
* 动态规划

### 答案
```
public int tribonacci(int n) {
    int[] f = new int[n + 1];
    if (n == 0) {
        return 0;
    }
    if (n == 1) {
        return 1;
    }
    if (n == 2) {
        return 1;
    }
    f[0] = 0;
    f[1] = 1;
    f[2] = 1;
    for (int i = 3; i <= n; i++) {
        f[i] = f[i - 1] + f[i - 2] + f[i - 3];
    }
    return f[n];
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(n)
