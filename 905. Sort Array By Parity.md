### 题目
https://leetcode.com/problems/sort-array-by-parity/

### 分类
* 双指针

### 答案
```
public int[] sortArrayByParity(int[] nums) {
    int lp = 0;
    int rp = nums.length - 1;
    while (lp < rp) {
        while (lp < nums.length - 1 && nums[lp] % 2 == 0) {
            lp++;
        }
        while (rp > 0 && nums[rp] % 2 != 0) {
            rp--;
        }
        if (lp < rp) {
            int tmp = nums[lp];
            nums[lp] = nums[rp];
            nums[rp] = tmp;
            lp++;
            rp--;
        }
    }
    return nums;
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(1)
