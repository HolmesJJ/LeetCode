### 题目
https://leetcode.com/problems/monotonic-array/

### 分类
* 通用

### 答案
```
public boolean isMonotonic(int[] nums) {

    boolean isAsc = false;
    boolean isDesc = false;
    for (int i = 0; i < nums.length - 1; i++) {
        if (nums[i] < nums[i + 1]) {
            isAsc = true;
        }
        if (nums[i] > nums[i + 1]) {
            isDesc = true;
        }
    }
    return !(isAsc && isDesc);
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(n)
