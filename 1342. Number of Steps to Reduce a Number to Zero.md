### 题目
https://leetcode.com/problems/number-of-steps-to-reduce-a-number-to-zero/

### 分类
* 通用

### 答案
```
public int numberOfSteps(int num) {
    int count = 0;
    while (num != 0) {
        if (num % 2 == 0) {
            num = num / 2;
        } else {
            num = num - 1;
        }
        count++;
    }
    return count;
}
```

### 复杂度
* 时间复杂度：O(logn)
* 空间复杂度：O(1)
