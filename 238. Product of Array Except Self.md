### 题目
https://leetcode.com/problems/product-of-array-except-self/

### 分类
* 通用

### 思路1
```
例：1 2 3 4 5 6 7 8

一次遍历计算一半
  2 3 4 5 6 7 8
1   3 4 5 6 7 8
1 2   4 5 6 7 8
1 2 3   5 6 7 8
1 2 3 4   6 7 8
1 2 3 4 5   7 8
1 2 3 4 5 6   8
1 2 3 4 5 6 7
```

### 答案
```
public int[] productExceptSelf(int[] nums) {

    int[] results = new int[nums.length];
    int num = 1;
    results[0] = 1;
    // 第一次遍历计算左半部分
    for (int i = 0; i < nums.length - 1; i++) {
        num = num * nums[i];
        results[i + 1] = num;
    }
    num = 1;
    // 第二次遍历计算右半部分
    for (int i = nums.length - 1; i > 0; i--) {
        num = num * nums[i];
        results[i - 1] = results[i - 1] * num;
    }
    return results;
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(n)
