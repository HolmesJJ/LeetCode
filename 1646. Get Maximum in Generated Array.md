### 题目
https://leetcode.com/problems/get-maximum-in-generated-array/

### 分类
* 动态规划

### 答案
```
public int getMaximumGenerated(int n) {
    if (n == 0) {
        return 0;
    }
    if (n == 1) {
        return 1;
    }
    int[] nums = new int[n + 1];
    nums[0] = 0;
    nums[1] = 1;
    int max = 1;
    for (int i = 1; 2 * i < n; i++) {
        nums[2 * i] = nums[i];
        nums[2 * i + 1] = nums[i] + nums[i + 1];
        max = Math.max(max, Math.max(nums[2 * i], nums[2 * i + 1]));
    }
    return max;
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(n)
