### 题目
https://leetcode.com/problems/longest-continuous-increasing-subsequence/

### 分类
* 通用

### 答案
```
public int findLengthOfLCIS(int[] nums) {

    int max = 1;
    int cur = 1;
    for (int i = 0; i < nums.length - 1; i++) {
        if (nums[i] < nums[i + 1]) {
            cur++;
        } else {
            max = Math.max(max, cur);
            cur = 1;
        }
    }
    max = Math.max(max, cur);
    return max;
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(n)
