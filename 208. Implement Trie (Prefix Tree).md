### 题目
https://leetcode.com/problems/implement-trie-prefix-tree/

### 分类
* 字典树Trie

### 思路1
* Trie中有一个根结点，根结点有26个子结点，每个子结点也有26个子结点，以此类推，其中每个结点代表一个字符
* `chars[i] - 'a'`代表使字符a的index从0开始，即b的index是1，c的index是2，以此类推

### 答案
```
class Trie {
    
    private class Node {
        Node[] childs = new Node[26];
        boolean isLeaf;
    }
    
    private final Node root;

    public Trie() {
        root = new Node();
    }
    
    public void insert(String word) {
        char[] chars = word.toCharArray();
        if (chars.length == 0) {
            return;
        }
        Node node = root;
        // 迭代储存word
        for (int i = 0; i < chars.length; i++) {
            if (node.childs[chars[i] - 'a'] == null) {
                node.childs[chars[i] - 'a'] = new Node();
            }
            if (i == chars.length - 1) {
                node.childs[chars[i] - 'a'].isLeaf = true;
            }
            node = node.childs[chars[i] - 'a'];
        }
    }
    
    public boolean search(String word) {
        char[] chars = word.toCharArray();
        if (chars.length == 0) {
            return true;
        }
        Node node = root;
        // 迭代搜索word
        for (int i = 0; i < chars.length; i++) {
            if (node.childs[chars[i] - 'a'] == null) {
                return false;
            }
            if (i == chars.length - 1 && !node.childs[chars[i] - 'a'].isLeaf) {
                return false;
            }
            node = node.childs[chars[i] - 'a'];
        }
        return true;
    }
    
    public boolean startsWith(String prefix) {
        char[] chars = prefix.toCharArray();
        if (chars.length == 0) {
            return true;
        }
        Node node = root;
        // 迭代搜索prefix
        for (int i = 0; i < chars.length; i++) {
            if (node.childs[chars[i] - 'a'] == null) {
                return false;
            }
            node = node.childs[chars[i] - 'a'];
        }
        return true;
    }
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(26^n)
