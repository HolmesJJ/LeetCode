### 题目
https://leetcode.com/problems/max-consecutive-ones/

### 分类
* 通用

### 答案
```
public int findMaxConsecutiveOnes(int[] nums) {
    
    int max = 0;
    int count = 0;
    for (int i = 0; i < nums.length; i++) {
        if (nums[i] == 1) {
            count++;
        } else {
            if (max < count) {
                max = count;
            }
            count = 0;
        }
    }
    if (max < count) {
        max = count;
    }
    return max;
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(n)
