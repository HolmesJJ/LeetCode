### 答案
```
public int[] countBits(int n) {

    int[] results = new int[n + 1];
    for (int i = 0; i <= n; i++) {
        results[i] = Integer.bitCount(i);
    }
    return results;
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(n)
