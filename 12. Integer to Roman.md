### 题目
https://leetcode.com/problems/integer-to-roman/

### 分类
* 通用

### 思路1
1. 罗马数字从左到右从大到小书写
2. 考虑4和9开头这类特殊情况

### 答案
```
public String intToRoman(int num) {

    String result = "";
    int carry = 1000;

    while (num > 0) {
        int digit = num / carry;
        String s = "";
        if (carry == 1000) {
            for (int i = 0; i < digit; i++) {
                s = s + "M";
            }
        } else if (carry == 100) {
            if (digit <= 3) {
                for (int i = 0; i < digit; i++) {
                    s = s + "C";
                }
            } else if (digit == 4) {
                s = s + "CD";
            } else if (digit >= 5 && digit <= 8) {
                s = s + "D";
                for (int i = 5; i < digit; i++) {
                    s = s + "C";
                }
            } else if (digit == 9) {
                s = s + "CM";
            }
        } else if (carry == 10) {
            if (digit <= 3) {
                for (int i = 0; i < digit; i++) {
                    s = s + "X";
                }
            } else if (digit == 4) {
                s = s + "XL";
            } else if (digit >= 5 && digit <= 8) {
                s = s + "L";
                for (int i = 5; i < digit; i++) {
                    s = s + "X";
                }
            } else if (digit == 9) {
                s = s + "XC";
            }
        } else if (carry == 1) {
            if (digit <= 3) {
                for (int i = 0; i < digit; i++) {
                    s = s + "I";
                }
            } else if (digit == 4) {
                s = s + "IV";
            } else if (digit >= 5 && digit <= 8) {
                s = s + "V";
                for (int i = 5; i < digit; i++) {
                    s = s + "I";
                }
            } else if (digit == 9) {
                s = s + "IX";
            }
        }
        num = num - digit * carry;
        carry = carry / 10;
        result = result + s;
    }

    return result;
}
```

### 复杂度
* 时间复杂度：O(n)
* 空间复杂度：O(n)
